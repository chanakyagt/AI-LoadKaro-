# LoadKaro Diagram Prompts

Each file in this folder contains a ready-to-paste prompt for generating a specific diagram.

## How to use

1. Open **Eraser.io** (https://app.eraser.io) — click "New diagram" → "Diagram from prompt"
2. Or open **Excalidraw** (https://excalidraw.com) for hand-drawn style
3. Or use **draw.io** (https://app.diagrams.net) for classic flowcharts
4. Copy the content from the relevant `.txt` file below
5. Paste it into the tool's prompt/import area
6. Export as PNG and save back here with the same name but `.png` extension
